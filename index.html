    Dashboard Focos de Queimadas 2020

    Esse projeto surge com a ideia de expandir e explorar os conhecimentos de Power BI e serviços AWS, analisando e visualizando dados de focos de queimadas das últimas 24 horas. Nesse projeto coletei os dados de queimadas através de dados públicos do INPE via API. A partir disso tratei os dados e armazenei o CSV no AWS S3 com a intenção de não ficar armazenando os dados no meu computador. Com o arquivo na nuvem e utilizando os serviços AWS Glue e Athena fiz a integração com o Power BI para fazer a análise de dados com gráficos. A integração permite que o dashboard seja atualizado facilmente a qualquer hora.

    Projeto PDF 2020

    Considerando a confidencialidade dos dados gerados na empresa e visando a segurança da informação, identifiquei a necessidade de mais transparência no processo de mesclagem das folhas dos relatórios, visto que utilizar um programa qualquer da internet que não se conhece o funcionamento gera riscos associados à segurança dos dados. A partir disso criei um programa simples em python que fazia a união das folhas dos relatórios. O Projeto PDF é uma aplicação web que faz essa atividade de juntar as folhas em PDF de forma simples e trás transparência para o processo, visto que os arquivos são armazenados temporariamente na nuvem e são apagados logo quando o processo é finalizado, sendo assim a confidencialidade dos dados é mantida preservada e soluciona um case da empresa sem custo adicional envolvido no processo. O serviço é uma aplicação Web cliente-servidor em que a interface foi feita com HTML, CSS e Javascript. O servidor foi escrito com Python e roda através do serviço AWS Lambda, o mesmo expôe uma API através do serviço AWS API Gateway que é utilizada pela interface.

    http://projeto-pdf.s3-website-us-east-1.amazonaws.com/
